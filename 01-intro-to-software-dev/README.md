## 📝 Key Takeaways

### Module 1.1: Getting Started

- Software developers combine technical skills (coding, problem-solving) with soft skills (communication, teamwork)
- The SDLC has 7 phases: Planning, Analysis, Design, Development, Testing, Deployment, Maintenance
- Java is robust, has a rich standard library, and strong community support

### Module 1.2: Variables & Operators

- Variables are named containers that store values with specific data types
- Java has primitive types (int, double, boolean, char) and reference types (String)
- Operators include arithmetic (+, -, \*, /), relational (>, <, ==), and logical (&&, ||, !)

- Strings are immutable sequences of characters backed by char arrays
- String manipulation is crucial for text processing and formatting
