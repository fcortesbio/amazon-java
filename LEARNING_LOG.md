### Week 4 (Oct 28 - Nov 3)

#### October 29, 2025
**Course**: Introduction to Software Development  
**Module**: 1.3 - Strings: Working with Text in Java

**Today's Focus**:
- Learned about arrays as containers for multiple values of the same type
- Understood that Strings in Java are immutable objects backed by char arrays
- Started exploring string manipulation concepts

**Key Insights**:
- String immutability provides security, concurrency, and performance benefits
- String methods create new objects rather than modifying originals

- Practice string manipulation exercises
